# kmz2txt

## Requirements
- Python 3
    - tqdm
    - requests
    - fastkml

## Installing
- 安裝 Python 3
- 將 `autokmz2txt.py` 放入一資料夾，其資料夾下必須含有 `KMZ`、`KML`、`TXT` 資料夾
- 在 `命令提示字元(cmd)`(Windows) 或 `終端機(Terminal)`(Linux/Mac) 下安裝套件： 
`pip3 install -r requirements.txt`
